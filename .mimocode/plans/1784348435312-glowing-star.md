- Missing `contentDescription` on ImageViews
- Font size `10sp` (below 11sp minimum)

## Files to Modify
1. **`app/src/main/res/values/strings.xml`** — add ~33 new string resources
2. **`app/src/main/res/layout/activity_multi_account_dashboard.xml`** — replace hardcoded values with `@string/` refs, add `contentDescription`, fix font size
