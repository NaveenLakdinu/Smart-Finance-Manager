# Fix: Student Dashboard Savings Amount Not Showing

## Root Cause

The student dashboard's savings widget loads from the **wrong source**:

| Component | Data Source | Status |
|-----------|-------------|--------|
| **SavingManagerActivity** | `users/{uid}/savings` subcollection | Works ✓ |
| **StudentDashboard.setupSavingsWidget()** | `users/{uid}/currentSavings` field | Broken ✗ |
| **StudentDashboard.loadAchievementData()** | `users/{uid}/savings` subcollection | Works ✓ |

The `currentSavings` field is not being set when users create savings goals. The actual data lives in the `savings` subcollection.

## Fix

Update `loadSavingsFromFirestore()` in `StudentDashboardActivity.java` to sum all savings from the `savings` subcollection instead of reading the `currentSavings` field.

### File to Modify
`app/src/main/java/com/example/smartfinancialmanagement/StudentDashboardActivity.java`

### Change
Replace `loadSavingsFromFirestore()` method (lines 228-247):

**Before:**
```java
private void loadSavingsFromFirestore(TextView txtValue) {
    FirebaseUser user = FirebaseAuth.getInstance().getCurrentUser();
    if (user == null) return;
    FirebaseFirestore.getInstance().collection("users").document(user.getUid())
            .get()
            .addOnSuccessListener(documentSnapshot -> {
                if (documentSnapshot.exists()) {
                    String currentSavings = documentSnapshot.getString("currentSavings");
                    if (currentSavings != null && !currentSavings.trim().isEmpty()) {
                        try {
                            double amt = Double.parseDouble(currentSavings.trim());
                            txtValue.setText(String.format(Locale.US, "LKR %.2f", amt));
                        } catch (NumberFormatException e) {
                            txtValue.setText("LKR " + currentSavings);
                        }
                    } else {
                        txtValue.setText("LKR 0.00");
                    }
                }
            });
}
```

**After:**
```java
private void loadSavingsFromFirestore(TextView txtValue) {
    FirebaseUser user = FirebaseAuth.getInstance().getCurrentUser();
    if (user == null) return;
    FirebaseFirestore.getInstance().collection("users").document(user.getUid())
            .collection("savings")
            .get()
            .addOnSuccessListener(querySnapshot -> {
                double totalSavings = 0;
                for (QueryDocumentSnapshot doc : querySnapshot) {
                    SavingModel saving = doc.toObject(SavingModel.class);
                    if (saving != null) {
                        totalSavings += saving.getCurrentAmount();
                    }
                }
                txtValue.setText(String.format(Locale.US, "LKR %.2f", totalSavings));
            })
            .addOnFailureListener(e -> txtValue.setText("LKR 0.00"));
}
```
